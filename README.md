To install and run the Demo OS (windows or linux) must install below

1- npm: 5.7.1
2- Angular CLI: 1.7.4
3- Node: 6.10.0

To run demo

1- extract the persons_managment.rar file
2- using terminal or cmd (go to directory persons managment)
3- run " npm install "
